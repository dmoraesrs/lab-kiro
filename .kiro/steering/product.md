# Product Vision - Lab Kiro

## Propósito

Projeto de laboratório para demonstrar o fluxo de desenvolvimento com **Kiro IDE + Claude Code + MCPs (Context7 + Serena)**. Serve como prova de conceito para apresentação na empresa.

## Público-Alvo

- Times de desenvolvimento da empresa
- Tech Leads avaliando adoção de AI-assisted development

## Objetivos

1. Demonstrar o fluxo completo: spec → código → review → testes → deploy
2. Validar os MCPs (Context7 + Serena) no dia a dia
3. Provar que guardrails automatizam qualidade
4. Servir como case para apresentação interna
